# Address Book Management System (C)

This project is a console-based Address Book application developed using C programming.
It allows users to manage contact details efficiently.

## Features
- Add new contacts
- Search contacts by name, phone, or email
- Edit existing contact details
- Delete contacts
- Sort contacts by name, phone number, or email
- Store data permanently using file handling

## Concepts Used
- Structures
- Pointers
- File I/O
- Modular programming

## How to Compile & Run
```bash
gcc *.c -o addressbook
./addressbook
